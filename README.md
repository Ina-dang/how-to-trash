# 🗑️ how-to-trash

> 우리동네 분리수거 가이드 🌱

지도에서 지역을 선택하면 그 동네의 쓰레기 배출 기준을 한눈에 확인할 수 있는 서비스입니다.

---

## ✨ 프로젝트 소개

how-to-trash는  
"이거 어디에 버리지?"  
라는 아주 현실적인 질문에서 시작된 서비스입니다

지역마다 다른 생활폐기물 배출 기준을  
지도 기반으로 쉽고 빠르게 확인할 수 있도록 돕습니다

현재는 카카오맵에서 구역을 선택하고,  
선택한 지역의 배출 기준 페이지로 이동하는 MVP 형태로 구현되어 있습니다

---

## 🤔 왜 만들었나

쓰레기 버릴 때 이런 경험 한 적 있지 않나요?

- "우리 동네는 음식물쓰레기 몇 시까지지?" ⏰
- "재활용은 어디에 내놔야 하지?" ♻️
- "오늘 종량제봉투 버려도 되나?" 🛍️
- 구청 사이트에 들어가도 정보가 여기저기 흩어져 있음 😵

그래서 만들었습니다!!

**지도에서 지역만 선택하면 바로 확인할 수 있는 서비스** ✨

---

## 💡 핵심 아이디어

이 서비스의 핵심은 단순합니다.

> ❌ 쓰레기 종류를 먼저 묻지 않는다  
> ✅ 지역을 먼저 선택하게 만든다

왜냐하면

- 같은 쓰레기라도 지역마다 기준이 다르고 📍
- 배출 시간, 장소, 방식도 다르고 🕒
- 결국 "어디서 버리느냐"가 먼저이기 때문입니다 🧩

---

## MVP 목표

### 포함

- 🗺️ 카카오맵 기반 지역 선택
- 📍 지역구 클릭
- 🧾 배출 기준 카드 보기
- 음식물쓰레기
- 생활쓰레기
- 재활용품
- 🕒 배출 시간 / 방법 / 장소 / 예외 안내
- 🔗 공식 출처 링크

### 아직 안 함

- 🔍 "딸기꼭지", "깨진 유리" 같은 품목 검색
- 🌏 전국 지역 지원
- 🤖 자동 데이터 수집
- 📍 현재 위치 자동 감지

---

## 🧪 현재 구현 범위

현재 홈 화면에서 지도 polygon으로 선택 가능한 구역은 아래 3곳입니다.

- 영등포구 🟢
- 양천구 🟢
- 관악구 🟢

현재 구현된 기능

- 카카오맵 기반 지역 선택 🗺️
- 구역 hover / click interaction 🖱️
- `/region/[regionKey]` 상세 페이지 📄
- 지역별 배출 기준 카드 표시 🧾
- 서울 자치구 생활쓰레기 배출 정보 데이터 정리 🧹

참고

- 서울 전체 데이터는 `src/data/seoul-waste-guides.ts`에 정리되어 있습니다 📦
- 다만 지도 경계 polygon은 현재 일부 구역만 연결되어 있습니다 ⚠️

---

## 🧭 사용자 흐름

```txt
서비스 접속
→ 지도 확인
→ 지역 선택
→ 배출 기준 페이지 이동
→ 생활쓰레기 / 음식물 / 재활용 정보 확인
```
