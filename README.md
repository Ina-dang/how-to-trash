# 🗑️ how-to-trash

> 우리동네 분리수거 가이드 🌱

지도에서 지역을 선택하면
그 동네의 쓰레기 배출 기준을 한눈에 확인할 수 있는 서비스입니다.

---

## 실행 방법

```bash
npm run dev
```

브라우저에서 `http://localhost:3000` 을 열면 됩니다.

---

## 프로젝트 소개

how-to-trash는
"이거 어디에 버리지?"
라는 아주 현실적인 질문에서 시작된 서비스입니다.

지역마다 다른 쓰레기 배출 기준을
지도 기반으로 쉽고 빠르게 확인할 수 있도록 돕습니다.

현재 MVP는
📍 서울특별시 영등포구 기준으로 동작합니다.

---

## 왜 만들었나

쓰레기 버릴 때 이런 경험 한 적 있지 않나요?

- ❓ "우리 동네는 음식물쓰레기 몇 시까지지?"
- ❓ "종량제 봉투 오늘 내놔도 되나?"
- ❓ "재활용 어디다 버리지?"
- 😵 구청 사이트 들어가면 정보가 너무 흩어져 있음

👉 그래서 만들었습니다.

**지도에서 지역만 선택하면 끝나는 서비스**

---

## MVP 목표

지금은 욕심 안 부립니다.

딱 이것만 합니다 👇

### 포함

- 🗺️ 카카오맵 기반 지역 선택
- 📍 영등포구 클릭
- 🧾 배출 기준 카드 보기
- 음식물쓰레기
- 생활쓰레기
- 재활용품
- 🕒 배출 시간 / 방법 / 장소 / 예외 안내
- 🔗 공식 출처 링크

### 아직 안 함

- 🔍 "딸기꼭지", "깨진 유리" 같은 품목 검색
- 🌏 전국 지역 지원
- 🤖 자동 데이터 수집
- 📍 현재 위치 자동 감지

---

## 핵심 아이디어

이 서비스의 핵심은 단순합니다.

> ❌ 쓰레기 종류를 먼저 묻지 않는다
> ✅ **지역을 먼저 선택하게 만든다**

왜냐하면 👇

- 같은 쓰레기라도 지역마다 기준이 다름
- 배출 시간, 장소, 방식도 다름
- 결국 "어디서 버리냐"가 먼저임

---

## 사용자 흐름

```txt
서비스 접속
→ 지도 확인
→ 영등포구 클릭
→ 배출 기준 페이지 이동
→ 음식물 / 생활 / 재활용 정보 확인
```
